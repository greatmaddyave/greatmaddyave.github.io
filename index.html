Responsive Design

Responsive design has never been more important in our generation. More and more people are viewing websites on mobile devices. To accomidate this growing trend technologies such as HTML5, CSS3 and Bootstrap make the transition from full webpages you see in a full browser to mobile a seamless and easy one. Go ahead and move the browser window to see responsive design in action

HTML5/CSS3 Dev

HTML is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and current version of the HTML standard. With this new standard brings new inline elements, dynamic page support as well as a new structure. If HTML is the frame then CSS3 is the decoration and structure. It's what gives a font its color and a font...well, it's font.

JavaScript jQuery

A healthy understanding of JavaScript is a standard requirement in any Web Developer. JavaScript is the backbone to the functionality, animation and even the backend of any and all modern web pages. jQuery is a library that I am also fairly confident in. Ask anyone who codes with JavaScript and jQuery and they'll agree, there's something magical about watching your code in action.
